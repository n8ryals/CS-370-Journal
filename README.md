# CS-370-Journal
**Pirate Intelligent Agent – Deep Q-Learning**
**Project Overview**
The goal of this project was to develop a deep Q-learning agent that can effectively navigate a maze, finding its way from the starting position (0,0) to the treasure located
in the bottom right corner, while avoiding obstacles and minimizing penalties. 
The environment (TreasureMaze.py) and replay memory class (GameExperience.py) were provided and my task was to implement the reinforcement learning logic 
in the Jupyter Notebook, which included:
•	Epsilon-greedy exploration/exploitation
•	Experience replay
•	Neural network training for Q-value approximation
•	Target network updates
•	Early stopping once stable performance was achieved
**What This Project Demonstrates**
Computer scientists design systems that solve complex problems through algorithms, data structures, and models. I utilized reinforcement learning combined with neural networks to build a system that can learn and adapt from experience instead of having the computer hard code routes. The agent was taught to navigate and conclude based on repeated interactions and rewards.
It also reflects how I approach problems as a computer scientist:
•	Break the system into components
•	Analyze behavior step-by-step
•	Debug systematically
•	Refine based on evidence
**Ethical Considerations**
Reinforcement learning systems depend heavily on reward design. Poor reward structures can produce unintended behavior. Projects must be approached with careful design, testing, and validation when building intelligent systems that may influence real-world decisions.
